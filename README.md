# Weather Dashboard Application

## Description
This is a Week 6 project demonstrating the use of external libraries in Python.
The application fetches real-time weather data from an online weather API using the requests library.

## Features
- Fetch current weather for any city
- Display temperature, humidity, wind speed, and conditions
- Error handling for invalid cities or network issues
- Interactive command-line interface

## Concepts Used
- External libraries (requests)
- HTTP requests
- JSON parsing
- Error handling
- Interactive CLI program

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python weather_app.py
